# Systems Programming Projects (C & C++)

This repository contains two core programming projects developed in C and C++, demonstrating both procedural and object-oriented programming paradigms. These projects highlight skills in data management, file handling, and software design.

---

## Project Overview

### Student Record Management System (C)
A command-line application designed to manage student records using structured data and persistent file storage.

- Built using C with a focus on procedural programming
- Utilizes structs to represent student data (ID, name, GPA)
- Implements file persistence using binary file I/O
- Supports adding, viewing, saving, and loading student records
- Demonstrates use of pointers, arrays, and modular functions

---

### Bank Account System (C++)
A banking simulation program designed using object-oriented programming principles.

- Developed in C++ using classes and objects
- Implements a base `BankAccount` class and a derived `CheckingAccount` class
- Supports deposits, withdrawals, and overdraft protection
- Uses inheritance and polymorphism (virtual functions)
- Demonstrates encapsulation and real-world system modeling
